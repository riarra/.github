# RIARRA

Международная логистика и грузоперевозки СНГ.

## Что мы делаем в этой организации

- **Pipeline автоматизации ставок** — агрегация грузовых тарифов из Telegram-каналов перевозчиков, парсинг, матчинг к заявкам клиентов
- **Dispatch Bot** — внутренняя тикет-система для диспетчеров (главный проект текущего квартала)
- **Интеграция ATI.SU** — автоматизация выгрузки грузов на крупнейшую площадку РФ
- **Сайт riarra.ru** — Next.js, SEO, блог
- **Внутренние боты** — @rilhom_bot (правая рука директора), @RiarraBot (рабочий бот)
